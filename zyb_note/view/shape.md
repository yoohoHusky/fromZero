#### shape各个属性
1. solid：填充

2. gradient：渐变
  1. startColor，endColor
  2. angle，渐变角度，45的整数倍
  3. type=“linear”，默认是现行渐变，
    “radial”，径向渐变需要制定半径gradientRadius="50"

3. stroke：描边
  1. width
  2. color
  3. dashWidth：横线宽度
  4. dashGap：隔开之间的距离

4. corners：圆角
  1. radius：圆角弧度
  2. toLeftRadius

5. padding：间隔

6. size
  1. width
  2. heigh
